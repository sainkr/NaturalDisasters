# 자연재해 예방 안전 서비스 _ 자연재해를 부탁해
## 개요
1. 작업 기간
+ 2020.07 ~ 

2. 소개
+ 자연재해 관련 속보를 실시간으로 보여주고, 관련 행동요령 및 대피소 위치 정보를 제공하는 자연재해 예방 안전 서비스



## 기능
+ 회원가입
+ 로그인, 자동로그인
+ 비밀번호 재발급
+ 문자 및 팝업 알림
  + 현재 위치 기반 자연재해 속보
  + 전국 자연재해 속보
  + 검색 지역 자연재해 속보
+ 한반도 위성사진 확인
+ 자연재해 정보
  + 행동요령 정보 제공
    + 호우, 낙뢰, 태풍, 폭설, 폭염, 지진
  + 대피소 위치 제공
    + 폭염, 지진
+ 119 긴급 전화
